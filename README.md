# 3D Digital Clock 

## Overview

This project demonstrates a 3D digital clock built using Three.js. The clock features a rotating second hand and a dark-themed background, showcasing basic 3D graphics and animations.

## Features

- **3D Clock Face**: A circular clock face rendered in 3D with a dark grey background.
- **Moving Second Hand**: A golden second hand rotates smoothly to represent the passing seconds.
- **Date**: Added date with day
- **Save Event function**: you can save event at view the logs
- **Responsive Design**: Adjusts to fit any screen size.

## Technologies Used

- HTML
- CSS
- JavaScript
